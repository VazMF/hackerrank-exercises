##Code

~~~
if __name__ == '__main__':
    n = int(input())
    
    # gera a lista
    lista = list()
    for c in range(0, n):
        lista.append(c)
        
    # printa o quadrado de cada indice
    for item in lista:
        print(item * item)
~~~
